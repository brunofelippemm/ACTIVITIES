Cloudiness

From the available data used to construct this plot, there is no evidence to support any correlation between cloudiness and latitude. But nevertheless, one can argue that compared to the number of existing cities the dataset used to make this plot have very few instances and therefore probably not statistically representative. So for further studies about a possible correlation between those variables, it is recommended to choose a larger dataset or append more instances to this one.
